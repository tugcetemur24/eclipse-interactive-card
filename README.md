# eclipse-interactive-card
Güneş tutulması temasından esinlenilmiş, modern ve interaktif bir 3D kart tasarımı.
Eclipse - Interactive UI Card
Bu proje, bir güneş tutulmasının (solar eclipse) mistik havasını modern web teknolojileriyle birleştiren interaktif bir kullanıcı arayüzü deneyimidir.

Özellikler
3D Flip Animasyonu: Kart üzerine tıklandığında yumuşak bir geçişle dönen yapı.

Glow (Parlama) Efektleri: CSS box-shadow ile oluşturulmuş tutulma efekti.

Responsive Tasarım: Farklı ekran boyutlarına uyumlu, ortalanmış düzen.

Modern CSS: perspective ve transform-style özellikleri ile derinlik algısı.
